# Student Study Habit Recommender System

A data-driven application that analyzes student study behavior and predicts academic performance to recommend optimized study strategies.

## Project Overview
This project uses exploratory data analysis, clustering techniques, and machine learning models to understand student study habits and provide personalized recommendations.

## Key Features
- Exploratory Data Analysis (EDA)
- Student Clustering using Machine Learning
- Performance Prediction
- Interactive Streamlit Dashboard
- Bulk and Individual Student Analysis

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- SQLite
- Plotly

## Project Structure
- Milestone-1: EDA and Data Preprocessing
- Milestone-2: Clustering and Prediction Models
- Milestone-3: Streamlit Dashboard Implementation
